# MTH308
The objective of this course project was to gain Practical Understanding of numerical techniques used in scientific computation and numerical analysis.

•Implemented Gaussian Elimination, LU Decomposition, Jacobi, and Gauss–Seidel methods for linear systems.


•Applied Bisection, Regula Falsi, Newton–Raphson, Secant, and Fixed Point Iteration methods for root finding.


•Implemented Hermite / Spline Interpolation, Newton’s Divided Differences, Trapezoidal and Simpson’s 1/3 Rule.
